Titre du projet: Tp final remise 3

But du projet:  créer un logiciel permettant la gestion d'une 
                entreprise de vente ou de location d'articles
                spécialisée

Date de remise: 18 décembre 2015

Comment utiliser le projet: Executer le "package" java dans le 
                            répertoire actuel et creer un instance 
                            de la classe principal


Auteur: Marc Plante

Instruction pour l'usager:

1)  Executer le "package" java dans le 
    répertoire actuel et creer un instance 
    de la classe principal
    
2)  L'inviter est présentement hors ligne il aura
    le choix de : lister des articles ou de se connecter
    en tant que utilissateur ou admin.
    
    ps. il faut un "id" et "mot de passe" valide pour
    pouvoir se connecter
    
    2-1)
    Dans le cas que l'inviter a choisi lister des articles,
    l'inviter pourra faire une liste des articles selon
    leur catégories.
    
    2-2)
    Dans le cas l'inviter ce connecte avec un id et un
    mot de passe valide, il pourra choisir entre :
    
    Afficher information:
        affiche les informations personelles de l'usager:
            Prnénom et nom, Coordonnée, id (numéro de membre)
            
    Afficher infomation article:
        affiche les données concernants les articles de l'utilisagers.
        
    Reserver article:
        reserve un article à un utilisateur et l'ajouter
        dans sa liste d'article
        
    Annuler une reservation:
        Annule la reservation d'un article
        
    Lister article:
        permet à l'utilissateur de lister des articles.
        
    2-3) (fonction supplémentaire des Admins)
    
        Si l'inviter c'est connecter sur un compte admin il peut
        faire les même requête que l'utilisateur, en choisisant quel
        utilisateur il veut faire la requete et en plus faire ces
        options:
        
        Ajouter et Supprimer des usagers
        
        Effectuer un pret pour un usager
        
        Annuler une réservation et effectuer un retour d'objet.
